## ✨ Sobre o Projeto

Este projeto tem como objetivo criar uma presença digital sofisticada para profissionais da área da beleza. A página foi desenhada para transmitir profissionalismo, confiança e elegância através de uma interface clara e elementos visuais de alta qualidade.

### 🎨 Principais Seções
- **Hero Section:** Destaque inicial com a proposta de valor, efeitos visuais premium e CTA claro para agendamento.
- **Portfólio em Carrossel:** Uma galeria de resultados que pausa ao interagir, exibindo transformações (Antes/Depois).
- **Serviços:** Cards com animações suaves (hover efeitos) detalhando as especialidades.
- **Diferenciais & Depoimentos:** Seções focadas em construir autoridade e prova social.

## 🚀 Tecnologias Utilizadas

- **React** para construção da interface estruturada e componentizada.
- **Vite** como build tool para um desenvolvimento rápido e otimizado.
- **Tailwind CSS** para estilização utilitária rápida, responsiva e moderna.
- **TypeScript** para garantir segurança de tipagem e manutenibilidade.

## 🛠️ Como executar o projeto localmente

**Pré-requisitos:** Node.js instalado na sua máquina.

1. **Clone o repositório:**
   ```bash
   git clone <url-do-repositorio>
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd alinezip
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Acesse no navegador:** \`http://localhost:5173\` (ou a porta informada no terminal).
